Experiencia de Proyectos

Aplicación Web de Recomendación de Lugares

Este proyecto consistió en el desarrollo de una aplicación web enfocada en ofrecer recomendaciones de lugares ideales para distintos tipos de salidas sociales, como citas, reuniones con amigos o actividades familiares. La plataforma fue diseñada con una interfaz dinámica e intuitiva, donde los usuarios podían consultar un listado de sitios organizados mediante un sistema de ranking que destacaba los más populares según la retroalimentación de la comunidad.

Además, se implementó un sistema de comentarios y reseñas en el que los visitantes compartían sus experiencias personales sobre los lugares visitados, aportando un valor agregado al permitir que otros usuarios tomaran decisiones mejor informadas. Desde un punto de vista técnico, el proyecto incluyó el manejo de bases de datos para almacenar las recomendaciones, un diseño adaptable para diferentes dispositivos y la estructuración de la información en secciones fáciles de navegar.

La principal complejidad de este proyecto radicó en integrar un flujo de interacción claro entre el usuario y la información, cuidando tanto la usabilidad como la visualización atractiva de los contenidos.

Plataforma Web de Cursos de Liderazgo

En este proyecto se diseñó y desarrolló una página web orientada a la promoción de cursos relacionados con el liderazgo y el crecimiento personal. El objetivo principal fue crear un espacio digital en el que los usuarios pudieran informarse de manera clara y organizada acerca de los programas de capacitación disponibles.

La plataforma estaba compuesta por varias secciones clave: un perfil del instructor, en el cual se describía su experiencia profesional y trayectoria; un apartado de contacto con los medios necesarios para resolver dudas o solicitar información adicional; y finalmente, un catálogo interactivo de cursos, en donde los interesados podían visualizar las clases ofrecidas junto con vistas previas de su contenido.

En términos técnicos, este proyecto implicó trabajar en la estructuración de información para hacerla accesible, aplicar principios de diseño web responsivo y asegurar que la navegación fuera sencilla y directa.

Sistema SIRELI con Tarjetas NFC

El proyecto SIRELI representó una experiencia innovadora que combinó el desarrollo de software con el uso de dispositivos físicos, específicamente tarjetas NFC. El objetivo era diseñar un sistema en el que dichas tarjetas funcionaran como llaves de acceso para abrir una aplicación vinculada, ofreciendo un ejemplo práctico de cómo la tecnología de comunicación inalámbrica de corto alcance puede integrarse con aplicaciones digitales.

El sistema estaba conformado por dos componentes principales: la aplicación, la cual respondía al acercar la tarjeta NFC al dispositivo compatible, y una página web complementaria que servía como plataforma de gestión y presentación del sistema. Esta integración permitió explorar conceptos de seguridad, compatibilidad entre hardware y software, así como la implementación de mecanismos de autenticación alternativos a los tradicionales.

La relevancia de este proyecto radicó en la oportunidad de trabajar con tecnologías emergentes aplicadas a soluciones prácticas, abriendo el camino hacia posibles usos en control de accesos, identificación digital y automatización de procesos.

Página Web de Venta de Drones de Rescate

Este proyecto académico consistió en el diseño de una página web conceptual enfocada en la venta de drones especializados en labores de rescate de montaña. La propuesta surgió como una idea innovadora dentro de la materia integradora, con el propósito de plantear un producto tecnológico capaz de brindar apoyo en situaciones de emergencia.

Los drones propuestos contaban con funcionalidades avanzadas, como cámaras integradas para la identificación de personas extraviadas, un botiquín de primeros auxilios incorporado para brindar atención inmediata y sistemas de apoyo destinados a montañistas en riesgo. Aunque el producto no fue desarrollado físicamente, la página cumplió la función de prototipo digital, mostrando cómo podría estructurarse la comercialización de un dispositivo de este tipo.

El sitio web incluía apartados de presentación del producto, características técnicas de los drones y un área de contacto para potenciales clientes interesados.

Calculadora de Ecuaciones Lineales

Como primer acercamiento al desarrollo web, se llevó a cabo la implementación de una página sencilla que funcionaba como calculadora de ecuaciones lineales. Aunque el nivel de complejidad fue bajo en comparación con proyectos posteriores, este trabajo resultó fundamental en el proceso de aprendizaje, ya que permitió poner en práctica los principios básicos de programación y lógica matemática aplicada a entornos digitales.

La calculadora recibía valores como parámetros de entrada, procesaba las operaciones correspondientes y entregaba el resultado de manera clara y directa al usuario. Este proyecto representó una oportunidad para trabajar con formularios, validación de datos y la integración de operaciones aritméticas dentro de un entorno web.

Más allá de su sencillez, el valor de este desarrollo estuvo en consolidar conocimientos iniciales de estructuración HTML, uso de estilos básicos en CSS y manejo de scripts para la interacción con el usuario.
